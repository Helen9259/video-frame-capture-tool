# 프레임 캡처

로컬 영상에서 원하는 순간을 캡처해 원하는 사이즈로 저장하는 업무용 웹 도구입니다.

- 서버 업로드 없이 브라우저 안에서만 처리됩니다 (영상 파일이 외부로 전송되지 않습니다).
- 로그인 없이 바로 사용할 수 있습니다.
- 정적 HTML 한 파일로 동작하며, 별도 빌드 과정이 없습니다.

## 사용법

`index.html`을 웹 서버에 올리거나 더블클릭으로 로컬에서 바로 열면 됩니다.

## GitHub Pages로 열람

이 저장소의 Settings → Pages에서 Source를 `Deploy from a branch`, Branch를 `main` / `(root)`로 설정하면
`https://<username>.github.io/video-frame-capture-tool/` 주소로 접속할 수 있습니다.
