# 썸네일 생성기

로컬 영상에서 프레임을 캡처하거나 이미지를 크롭해, 원하는 사이즈로 저장하는 업무용 웹 도구입니다.

- 서버 업로드 없이 브라우저 안에서만 처리됩니다 (영상·이미지 파일이 외부로 전송되지 않습니다).
- 로그인 없이 바로 사용할 수 있습니다.
- 정적 HTML 한 파일로 동작하며, 별도 빌드 과정이 없습니다.
- 영상 프레임 캡처와 이미지 크롭을 하나의 작업 화면에서 함께 다룰 수 있고, 저장 사이즈 프리셋은 두 작업에서 공통으로 재사용됩니다.
- 결과 화면에서 파일명 일괄 변경(찾기/바꾸기), 중복 파일명 표시, 파일별/전체 확장자(PNG·JPG) 변경, 개별/ZIP 다운로드를 지원합니다.

## 사용법

`index.html`을 웹 서버에 올리거나 더블클릭으로 로컬에서 바로 열면 됩니다.

## GitHub Pages로 열람

이 저장소의 Settings → Pages에서 Source를 `Deploy from a branch`, Branch를 `main` / `(root)`로 설정하면
`https://<username>.github.io/video-frame-capture-tool/` 주소로 접속할 수 있습니다.
